HR Risk Decision Support System
##  Overview
This repository contains the coursework for **Week 2 Lab 29**, focusing on building an HR **Risk Decision Support System (DSS)**.  
The project uses employee data (department, age, experience, satisfaction, attendance, overtime, performance) to classify risk levels and generate actionable HR recommendations.

---

##  Objectives
- Create and analyze a synthetic HR dataset  
- Classify employees into **High, Medium, or Low Risk** categories  
- Generate decision support recommendations for HR teams  
- Perform department-wise risk distribution analysis  
- Provide business insights based on employee satisfaction and overtime  

---

##  Tools & Libraries Used
- **Python 3**  
- **Pandas** – for dataset creation, handling, and analysis  
- **NumPy** – for numerical operations and random data generation
